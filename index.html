高级设置

本工具不需要任何 API 密钥。唯一可选项是「链接可用性探测代理」，用于检测镜像链接是否可访问。默认使用公开 CORS 服务，你也可以换成自己的代理。

程序会把待检测的链接 encodeURIComponent 后拼接在该地址末尾，并期望返回包含 status.http_code 的 JSON。留空则禁用可用性探测。
